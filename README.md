## 🚀 Projects I'm Working On

- **[TEKNOFEST Çelikkubbe](#)** — Developing an AI-based Close-In Weapon System (CIWS) for the TEKNOFEST 2026 Air Defense Systems Competition. Designing algorithms for autonomous target detection, tracking, and friend-or-foe identification using PyTorch and OpenCV, with route optimization for swarm attacks and secure real-time communication between the control unit and UI.
- **[MVC_Test_Orchestrator_V1.2](https://github.com/ilaydim/MVC_Test_Orchestrator_V1.2)** — A multi-agent system (9 specialized agents) that automates extraction and validation of Model-View-Controller architectures from Software Requirements Specification documents. Includes a RAG pipeline with vector-based search to keep architectural decisions grounded in documented requirements, plus a static-analysis audit tool (Python `ast`) that gives real-time natural-language feedback in VS Code. Evaluated with an LLM-as-a-Judge framework. **Published at IISEC 2026.**
- **[idas-project (IDAS)](https://github.com/ilaydim/idas-project)** — An intelligent documentation assistant built in collaboration with HAVELSAN SUIT to improve Software Requirements Specification quality: real-time terminology consistency checks, missing-section detection, AI-driven draft requirement generation following HAVELSAN standards, automated requirement extraction from .docx files, classification, and ambiguity/untestability/conflict analysis.
- **[rf_classification](https://github.com/ilaydim/rf_classification)** — A Hybrid CNN-Transformer architecture for Automatic Modulation Classification (AMC) in electronic warfare. CNN extracts local I/Q signal features while Transformer attention captures long-range temporal dependencies, trained on the DeepSig RadioML 2018.01A benchmark across 24 modulation types and 26 SNR levels, with ablation studies on robustness at low SNR.
